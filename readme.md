# Тестовое задание Ростелеком

## Описание проекта
Проект представляет собой парсер тарифных планов. Программа собирает информацию о доступных тарифах и сохраняет ее в Excel-файл.

## Установка

### 1. Установите Python и Poetry
    - Установите Python 3.14 или выше с официального сайта: [python.org](https://python.org)
    - Установите Poetry с официального сайта: [Poetry](https://python-poetry.org/)

### 2. Клонирование репозитория
```
git clone
```

### 3. Установка зависимостей
```
cd rtk-test-task
poetry install
```

## Запуск
### Запустите из корневой директории проекта
```
cd rtk-test-task
poetry run python -m src.main
```

## Результат работы
После выполнения программы в папке output будет создан файл rialcom_tariffs.xlsx с таблицей тарифов.
